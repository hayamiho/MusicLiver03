# MusicLiver05 プロジェクト憲法 (ANTIGRAVITY.md)

## 1. 目的・ターゲット
- **目的**: 路上ライブアーティストへの支援（投げ銭）を円滑にするシステム。
- **ターゲット**: 路上ライブの視聴者、ファン。
- **利用シーン**: ライブ中、配信中、イベント前後。

## 2. 技術スタック
- **Frontend**: HTML, Vanilla CSS, JavaScript.
- **Backend/Hosting**: Vercel (URL: https://music-liver-05.vercel.app/).
- **認証**: パスワード保護 (music03).

## 3. MVPの範囲
- インデックスページからのスムーズな会員登録誘導。
- ライブスケジュール確認機能。
- 現在演奏中のアーティストのリアルタイム視聴・支援機能。
