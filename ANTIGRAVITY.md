# MusicLiver05 プロジェクト憲法 (ANTIGRAVITY.md)

## 1. 目的・ターゲット
- **目的**: 路上ライブアーティストへの支援（投げ銭）を円滑にするシステム。
- **ターゲット**: 路上ライブの視聴者、ファン。
- **利用シーン**: ライブ中、配信中、イベント前後。

## 2. 技術スタック
- **Frontend**: HTML, Vanilla CSS, JavaScript.
- **Backend/Hosting**: Vercel (URL: https://music-liver-05.vercel.app/).
- **認証**: パスワード保護 (music03).

## 3. MVPの範囲
- インデックスページからのスムーズな会員登録誘導。
- ライブスケジュール確認機能。
- 現在演奏中のアーティストのリアルタイム視聴・支援機能。

## 5. 新規LP (Waitlist) の要件
- **目的**: MLSSのアーティスト・事務所向け宣伝・事前登録LP。
- **配置先**: `/waitlist/index.html`
- **デザイン方針**:
  - ベースは明るくポップな構成（ラフ案ベース）。
  - 色使いは現行MLSSに完全に寄せる（白基調、ネイビー文字 `#01255a`、ピンク〜青のグラデーションボタン）。
  - 背景画像（`01.jpg`〜`15.jpg`）は、特定の文章に紐づけるのではなく、画面全体に「フワッと」配置するアンビエント（環境）演出として使用する。
- **構成**:
  - ヒーローセクション（スマホモックアップ `phone.png` ＋ 登録ボタン）
  - 課題とメリット（テキスト主体のジグザグ配置、背景に画像を散りばめる）
  - 利用シーンと導入の流れ
  - 最終CTA（登録ボタン -> Googleフォームへ）
